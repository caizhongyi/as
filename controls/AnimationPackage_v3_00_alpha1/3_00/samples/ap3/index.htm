AnimationPackage 3 is an ActionScript 3 library compatible to Flash Player 9. It helps you to create powerful, maintainable animations and primitive shapes in an efficient and easy way, resulting in small file sizes.

Like all versions of AnimationPackage, this version 3 is independent of the Flex 2 Framework. If you are able to use the Flex 2 Framework, I recommend you consider using its Flex 2 Effects Framework API or extentions of it, instead of AnimationPackage.
